## Economy Interfaces

### Profile

#### Information Shown vs Hidden

**SHOWN (Always)**

- Account status indicator
- Level + XP progress bar
- Primary balance (coins)
- Equipped title/badges (if any)
- Quick stats: Rep, Bank total, Inventory count

**SHOWN (On Demand)**

- Full currency breakdown (via `/balance`)
- Full inventory (via `/inventory`)
- Achievement progress (via `/achievements`)
- Perk levels (via `/perks list`)

**HIDDEN**

- Internal IDs
- Exact timestamps (show relative)
- Debug/moderation info
